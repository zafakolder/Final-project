# Проект Mesto

Учебный проект от Яндекс.Практикума — интерактивная страница для обмена фотографиями. Реализовано подключение к серверному API, добавление, удаление, лайки карточек, редактирование профиля и аватара.

## 🚀 Функциональность

- Загрузка карточек и данных пользователя с сервера.
- Добавление новых карточек с валидацией формы.
- Удаление карточек (только своих) с подтверждением.
- Лайки карточек с обновлением счётчика.
- Редактирование профиля (имя, описание).
- Обновление аватара.
- Модальное окно со статистикой карточки (дата, автор, список лайкнувших).
- Адаптивная вёрстка, валидация всех форм.
- Закрытие модальных окон по оверлею, Escape и крестику.

## 🛠 Технологии

- HTML, CSS (flexbox, grid, медиазапросы)
- JavaScript (ES6+)
- Vite (сборка)
- API (REST)
- GitHub Pages (деплой)

## 🔧 Команды для разработки

| Команда            | Действие                                                                 |
|--------------------|--------------------------------------------------------------------------|
| `npm install`      | Установить все зависимости                                               |
| `npm run dev`      | Запустить локальный сервер разработки с горячей перезагрузкой           |
| `npm run build`    | Собрать проект в папку `dist` для продакшена                            |
| `npm run deploy`   | Собрать и опубликовать проект на GitHub Pages                           |
